Instalace Hermes Agent

Kompletní návod pro Linux, macOS, WSL2 a Windows. Zprovozněte svého AI agenta za 60 sekund.

Rychlá instalace

Jeden příkaz — instalátor se postará o vše:

# Linux / macOS / WSL2 / Android (Termux) $ curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash
# Windows (PowerShell) PS> iex (irm https://hermes-agent.nousresearch.com/install.ps1)

Co instalátor udělá

Instalátor automaticky nainstaluje všechny závislosti:

  • ✅ uv — rychlý Python package manager
  • ✅ Python 3.11 — přes uv, bez sudo
  • ✅ Node.js v22 — pro automatizaci prohlížeče a WhatsApp
  • ✅ ripgrep — rychlé vyhledávání v souborech
  • ✅ ffmpeg — konverze audia pro TTS
  • ✅ Git repozitář — klonování a nastavení
  • ✅ Globální příkaz hermes

Poznámka: Nemusíte instalovat Python, Node.js, ripgrep ani ffmpeg ručně. Instalátor detekuje co chybí a nainstaluje to automaticky. Jediný předpoklad je Git (git --version). Na Linuxu se ujistěte, že máte curl a xz-utils.

Po instalaci

Obnovte shell a spusťte:

# Obnovení shellu $ source ~/.bashrc # nebo: source ~/.zshrc # Spuštění $ hermes

Konfigurace

# Vyberte LLM provider a model $ hermes model # Konfigurace nástrojů $ hermes tools # Nastavení messaging platforem (Telegram, Discord…) $ hermes gateway setup # Nebo vše najednou přes průvodce $ hermes setup

Nejrychlejší cesta: Nous Portal

Jedno předplatné pokrývá 300+ modelů plus Tool Gateway (web search, generování obrázků, TTS, cloud browser).

$ hermes setup --portal

Instalace pro systémové služby

Pro spuštění Hermes jako dedikovaného uživatele (např. systemd služba):

  1. Správce: Instalace systémových knihoven pro Chromium

    $ sudo npx playwright install-deps chromium

  2. Servisní uživatel: Instalace Hermes

    $ curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash

    Pokud nepotřebujete automatizaci prohlížeče:

    $ curl -fsSL .../install.sh | bash -s -- --skip-browser

  3. Dostupnost příkazu hermes

    # Přidání do PATH $ echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc # Nebo symlink (jako admin) $ sudo ln -s /home/hermes/.hermes/hermes-agent/venv/bin/hermes /usr/local/bin/hermes

  4. Ověření

    $ hermes doctor

Desktop instalátor

Na macOS a Windows můžete použít Hermes Desktop installer — stáhněte z hermes-agent.nousresearch.com a spusťte. Po instalaci běží hermes desktop.

Ruční / Vývojářská instalace

Pro přispěvatele nebo plnou kontrolu nad virtuálním prostředím:

# Klonování repozitáře $ git clone https://github.com/NousResearch/hermes-agent.git $ cd hermes-agent # Vytvoření virtuálního prostředí $ python3 -m venv venv $ source venv/bin/activate # Instalace závislostí $ uv pip install -e .

Minimální požadavky

  • 💾 RAM: 1 GB min, 2 GB doporučeno
  • ⚙️ CPU: 1 vCPU minimum
  • 💿 Úložiště: 10–20 GB SSD
  • 🐧 OS: Ubuntu 22.04+ nebo moderní Linux
  • 🤖 Model: Vlastní API klíč — Claude, OpenAI, Gemini, nebo lokální

Žádná GPU není potřeba — Hermes volá externí API.
