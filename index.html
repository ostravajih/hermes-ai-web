Základy

Co je Hermes Agent?

Open-source autonomní AI agent od Nous Research, vydaný v únoru 2026. Běží na Linuxu, macOS a WSL2 — bez závislostí, vše nainstaluje jedním příkazem.

🧠

Persistentní paměť

Pamatuje si vaše preference, projekty a prostředí napříč všemi relacemi. Čím déle běží, tím lépe vás zná — žádné opakované vysvětlování kontextu.

📡

Multi-platformní přístup

Telegram, Discord, Slack, WhatsApp, Signal, Matrix, Email, SMS — vše z jedné gateway. Začněte na Telegramu, pokračujte v terminálu.

⚡

Automatické dovednosti

Když Hermes vyřeší složitý problém, vytvoří znovupoužitelnou dovednost (skill). Kompatibilní s otevřeným standardem agentskills.io.

⏰

Autonomní plánování

Vestavěný cron plánovač — ranní briefingy, PR review, monitoring testů, blog watchery. Běží, i když spíte.

🔧

47+ vestavěných nástrojů

Web search, prohlížeč, terminál, spouštění kódu, analýza obrázků, generování obrázků, TTS, delegování sub-agentů a další.

🔒

Bezpečnost a soukromí

Žádná telemetrie, žádné sledování. Všechna data zůstávají na vašem serveru ve složce ~/.hermes/. MIT licence — auditovatelný kód.

Proč Hermes

Tři pilíře, které ho odlišují

Většina nástrojů je skvělá v daném okamžiku, ale slabá v průběhu času. Hermes je jiný.

🧠

Paměť, která se kumuluje

Vrstvený paměťový systém: uživatelský profil, paměť agenta, dovednosti a historie relací. Vše uloženo lokálně jako čitelné markdown soubory. Přežije každý restart a výměnu modelu.

⏰

Autonomní plánování

Vestavěný cron plánovač s doručováním na jakoukoliv platformu. Ranní briefingy na Telegram, automatické PR review, monitoring testů — vše bez vaší přítomnosti.

📡

Dostupný odkudkoliv

Stejný agent, stejná paměť, různé platformy. Přepínejte mezi Telegramem, Discordem, terminálem nebo Web UI — kontext zůstává zachován.

Rychlý start

Zprovozněte za 60 sekund

Žádné závislosti. Jeden příkaz a Hermes běží.

# Instalace na Linux, macOS nebo WSL2 $ curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash # Konfigurace LLM providera $ hermes setup # Spuštění $ hermes # Nastavení messaging gateway (volitelné) $ hermes gateway setup $ hermes gateway

Podporuje: Nous Portal, OpenRouter, OpenAI, Anthropic, Google, DeepSeek a jakýkoliv OpenAI-kompatibilní endpoint.
