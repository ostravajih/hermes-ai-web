Basics

What is Hermes Agent?

An open-source autonomous AI agent by Nous Research, released in February 2026. Runs on Linux, macOS, and WSL2 — no dependencies, installs with a single command.

🧠

Persistent Memory

Remembers your preferences, projects, and environment across all sessions. The longer it runs, the better it knows you — no more repeating context.

📡

Multi-Platform Access

Telegram, Discord, Slack, WhatsApp, Signal, Matrix, Email, SMS — all from one gateway. Start on Telegram, continue in terminal.

⚡

Automatic Skills

When Hermes solves a complex problem, it creates a reusable skill. Compatible with the open agentskills.io standard.

⏰

Autonomous Scheduling

Built-in cron scheduler — morning briefings, PR reviews, test monitoring, blog watchers. Runs even while you sleep.

🔧

47+ Built-in Tools

Web search, browser, terminal, code execution, image analysis, image generation, TTS, sub-agent delegation, and more.

🔒

Security & Privacy

No telemetry, no tracking. All data stays on your server in ~/.hermes/. MIT license — auditable code.

Why Hermes

Three pillars that set it apart

Most tools are great in the moment but weak over time. Hermes is different.

🧠

Cumulative Memory

Layered memory system: user profile, agent memory, skills, and session history. All stored locally as readable markdown files. Survives every restart and model change.

⏰

Autonomous Scheduling

Built-in cron scheduler with delivery to any platform. Morning briefings on Telegram, automatic PR reviews, test monitoring — all without your presence.

📡

Available Anywhere

Same agent, same memory, different platforms. Switch between Telegram, Discord, terminal, or Web UI — context stays preserved.

Quick Start

Get up and running in 60 seconds

No dependencies. One command and Hermes is running.

# Install on Linux, macOS, or WSL2 $ curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash # Configure LLM provider $ hermes setup # Run $ hermes # Set up messaging gateway (optional) $ hermes gateway setup $ hermes gateway

Supports: Nous Portal, OpenRouter, OpenAI, Anthropic, Google, DeepSeek, and any OpenAI-compatible endpoint.
