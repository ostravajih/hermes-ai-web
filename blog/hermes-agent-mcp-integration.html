Hermes Agent a MCP

10. července 2026 · 3 min

MCP (Model Context Protocol) je otevřený standard od Anthropicu pro připojení AI agentů k externím nástrojům, datům a službám. Hermes Agent MCP plně podporuje — jako klient i server.

Co MCP umožňuje

  • 🔌 Připojení Hermes k libovolnému MCP serveru (databáze, API, souborové systémy)
  • 🛠️ Automatické objevení nástrojů, které server nabízí
  • 🔄 Hermes jako MCP server pro jiné agenty (Claude Code, Cursor, Codex)
  • 📦 Instalace MCP serverů z katalogu

Přidání MCP serveru k Hermes

# Z URL (HTTP/SSE transport) $ hermes mcp add my-server --url https://example.com/mcp # Z příkazu (stdio transport) $ hermes mcp add local-tools --command "python3 my_mcp_server.py" # Instalace z katalogu $ hermes mcp install @anthropic/fetch $ hermes mcp install @modelcontextprotocol/server-filesystem

Hermes jako MCP server

Spusťte Hermes v režimu MCP serveru — ostatní agenti (Claude Code, Cursor) k němu mohou přistupovat:

$ hermes mcp serve

Potom v Claude Code nebo Cursor přidejte Hermes jako MCP server:

# V .cursor/mcp.json nebo claude_desktop_config.json: { "mcpServers": { "hermes": { "command": "hermes", "args": ["mcp", "serve"] } } }

Takto můžete používat Claude Code na kódování, zatímco Hermes běží na pozadí a spravuje paměť, plánování, cron úkoly a notifikace.

Správa MCP serverů

$ hermes mcp list # Seznam připojených serverů $ hermes mcp test NAME # Otestovat připojení $ hermes mcp configure NAME # Zapnout/vypnout konkrétní nástroje $ hermes mcp remove NAME # Odstranit server

Praktický příklad: MCP server pro soubory

# Instalace filesystem MCP serveru $ npx @anthropic/mcp-server-filesystem /cesta/k/projektu # Přidání k Hermes $ hermes mcp add filesystem --command "npx @anthropic/mcp-server-filesystem /cesta/k/projektu" # Hermes teď může číst a zapisovat soubory v daném adresáři
🔌 MCP katalog: github.com/modelcontextprotocol/servers — oficiální a komunitní MCP servery pro různé služby (Google Drive, Slack, PostgreSQL, Puppeteer a desítky dalších).
