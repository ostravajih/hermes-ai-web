Hermes Agent na macOS

10. července 2026 · 5 min

Hermes Agent běží nativně na macOS — Apple Silicon (M1/M2/M3/M4) i Intel. Instalace trvá minutu, desktop aplikace poskytuje plnohodnotné grafické rozhraní.

Rychlá instalace (shell skript)

$ curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash

Instalátor automaticky nainstaluje Python 3.11, Node.js, ripgrep, ffmpeg a všechny závislosti.

Alternativní instalace přes Homebrew

# Pokud nemáte Homebrew: $ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)" # Instalace Hermes Agent $ brew install hermes-agent

Konfigurace po instalaci

$ source ~/.zshrc # nebo: source ~/.bashrc $ hermes setup # Průvodce konfigurací (model, provider, API klíč) $ hermes # Spuštění v terminálu

Desktop aplikace

Hermes má nativní Electron desktop aplikaci pro macOS:

$ hermes desktop

Klíčové funkce desktop aplikace:

  • Streaming chat — odpovědi se zobrazují v reálném čase
  • Drag & drop souborů — přetáhněte obrázky, PDF, kód
  • Cmd+K paleta — rychlé příkazy jako ve VS Code
  • Session list — přehled všech konverzací
  • Nativní notifikace — upozornění když agent dokončí úkol
  • Live subagent watch-windows — sledování paralelních agentů
  • Přepínání modelů — výběr providera přímo z UI

Apple Silicon: Výhody pro AI

Macy s čipy M1-M4 mají sdílenou paměť (unified memory) — CPU i GPU sdílí stejnou RAM. To znamená, že:

  • 16GB MacBook Air může provozovat lokální LLM modely až do velikosti 13B parametrů
  • Ollama na Macu automaticky využívá Metal (GPU akceleraci) — modely běží výrazně rychleji než na Intel Macích
  • Žádné externí GPU není potřeba

Spuštění lokálního LLM na macOS

# Instalace Ollama $ brew install ollama # Spuštění Ollama služby na pozadí $ brew services start ollama # Stažení modelu $ ollama pull llama3.1 # Konfigurace v Hermes: Custom provider → http://localhost:11434/v1

Integrace s macOS

  • Terminal.app / iTerm2 / Warp: Hermes funguje v jakémkoliv terminálu
  • Spotlight: Můžete si vytvořit alias pro rychlé spuštění
  • Menu bar: Desktop aplikace se minimalizuje do menu baru
  • Touch ID: Pro sudo příkazy můžete použít Touch ID

Řešení problémů na macOS

"hermes: command not found"

Instalátor přidá Hermes do PATH. Pokud nefunguje, přidejte ručně: export PATH="$HOME/.local/bin:$PATH" >> ~/.zshrc

Gatekeeper blokuje spuštění

macOS může blokovat nepodepsané aplikace. Při prvním spuštění hermes desktop jděte do System Settings → Privacy & Security → Allow Anyway.

Ollama nevidí GPU

Na Apple Silicon je GPU akcelerace automatická. Ověřte: ollama run llama3.1 --verbose a zkontrolujte, zda výstup ukazuje "Metal" jako backend.
