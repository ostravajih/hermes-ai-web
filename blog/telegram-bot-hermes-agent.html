Telegram bot s Hermes Agent

8. července 2026 · Doba čtení: 4 minuty

Telegram je nejoblíbenější platforma pro Hermes Agent. Během 5 minut můžete mít vlastního AI agenta, který odpovídá na zprávy, spouští úkoly a pamatuje si kontext napříč konverzacemi.

Krok 1: Vytvoření bota přes BotFather

Otevřete Telegram a napište @BotFather:

/newbot → Zadejte jméno: Můj Hermes → Zadejte username: MujHermesBot → Uložte si HTTP API token!

Krok 2: Konfigurace Hermes gateway

Na vašem VPS spusťte:

$ hermes gateway setup → Vyberte Telegram → Vložte token z BotFather

Krok 3: Spuštění

$ hermes gateway

Bot je online! Napište mu na Telegramu.

Co všechno Telegram bot umí

  • 💬 Chat — plnohodnotná konverzace s AI agentem
  • 🔧 Všechny nástroje — web search, spouštění kódu, generování obrázků
  • 🧠 Persistentní paměť — pamatuje si vás napříč dny i týdny
  • 🎤 Hlasové zprávy — automatický přepis a odpověď
  • ⏰ Cron plánovač — ranní briefingy přímo do Telegramu
  • /slash příkazy — /new, /model, /skills a další
🔒 Bezpečnost: V souboru ~/.hermes/.env můžete nastavit TELEGRAM_ALLOWED_USERS — čárkou oddělené ID uživatelů, kteří můžou s botem komunikovat. Tím zabráníte neoprávněnému přístupu.

Nastavení ranního briefingu

Nejužitečnější funkce — každé ráno vám Hermes pošle souhrn:

/cron create "0 8 * * *" "Dej mi ranní briefing: počasí, GitHub notifikace, dnešní úkoly"
