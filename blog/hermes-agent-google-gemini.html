Hermes Agent s Google Gemini

10. července 2026 · 3 min

Google Gemini nabízí 1 500 požadavků denně zdarma — ideální pro začátek s Hermes Agent. Navíc má výbornou podporu češtiny.

Získání API klíče

  1. Jděte na aistudio.google.com
  2. Přihlaste se Google účtem
  3. Vlevo: Get API Key → Create API Key
  4. Zkopírujte klíč

Konfigurace

# ~/.hermes/.env GEMINI_API_KEY=AIza... # nebo: $ hermes model → Provider: Google → Model: gemini-2.5-flash

Free tier limity

  • 1 500 požadavků/den
  • 1 000 000 tokenů/minuta
  • Dostačující pro běžné použití
🇨🇿 Gemini má jednu z nejlepších podpor češtiny ze všech modelů. Odpovídá gramaticky správně a přirozeně. Pro českou komunitu je to skvělá volba.
