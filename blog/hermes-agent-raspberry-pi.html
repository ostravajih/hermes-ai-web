Hermes Agent na Raspberry Pi

10. července 2026 · 3 min

Raspberry Pi 4/5 s 4-8 GB RAM je dostačující pro Hermes Agent. Spotřeba ~5W, cena ~1 000 Kč — ideální pro domácí 24/7 AI agenta.

Hardware

  • Raspberry Pi 4 (4 GB) — minimum, ~800 Kč
  • Raspberry Pi 5 (8 GB) — doporučeno, ~1 500 Kč
  • Externí SSD — povinné (SD karta se rychle opotřebí)

Instalace

# Raspberry Pi OS (64-bit) na SSD $ sudo apt update && sudo apt upgrade -y $ curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash

Optimalizace

# Vypnutí swapu (šetří SSD) $ sudo dphys-swapfile swapoff && sudo dphys-swapfile uninstall # Přesun logů do RAM (šetří SSD) $ sudo nano /etc/fstab # Přidejte: tmpfs /var/log tmpfs defaults,noatime 0 0

Doporučené nastavení

Použijte cloudové API (DeepSeek, Gemini Flash) — Raspberry Pi nemá výkon na lokální LLM.

💡 Ideální pro: Telegram bota, domácí automatizaci, monitoring serveru. Nevhodné pro: lokální LLM, náročné výpočty.
