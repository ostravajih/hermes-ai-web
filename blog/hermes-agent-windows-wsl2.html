Hermes Agent na Windows a WSL2

8. července 2026 · Doba čtení: 5 minut

Hermes Agent běží nativně na Linuxu a macOS. Na Windows ho zprovozníte přes WSL2 (Windows Subsystem for Linux) — oficiální virtualizační vrstvu od Microsoftu, která je zdarma a integrovaná přímo do Windows 10/11.

Krok 1: Instalace WSL2

Otevřete PowerShell jako administrátor:

PS> wsl --install

Tento příkaz nainstaluje WSL2 a výchozí distribuci Ubuntu. Po dokončení restartujte počítač.

Krok 2: Nastavení Ubuntu

Po restartu se spustí Ubuntu terminál. Vytvořte si uživatelské jméno a heslo. Pak aktualizujte systém:

$ sudo apt update && sudo apt upgrade -y

Krok 3: Instalace Hermes Agent

$ curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash $ source ~/.bashrc $ hermes setup
⚡ Výhoda WSL2: Hermes běží v Linuxovém prostředí, ale máte přístup k souborům Windows přes /mnt/c/. Můžete tak pracovat na projektech uložených ve Windows.

Řešení problémů

WSL2 se nenainstaluje

Ujistěte se, že máte zapnutou virtualizaci v BIOSu (Intel VT-x / AMD-V).

Hermes nevidí GPU

Pro lokální LLM přes Ollama potřebujete WSL2 s podporou GPU: wsl --update a pak v Ubuntu ollama serve.
