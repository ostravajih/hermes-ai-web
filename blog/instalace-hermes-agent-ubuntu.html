Jak nainstalovat Hermes Agent na Ubuntu

8. července 2026 · Doba čtení: 5 minut

Hermes Agent je open-source AI agent od Nous Research, který běží na vašem serveru. Na rozdíl od cloudových řešení máte plnou kontrolu nad daty, pamětí a konfigurací. V tomto návodu si ukážeme, jak ho zprovoznit na Ubuntu VPS za méně než minutu.

Co budete potřebovat

  • VPS s Ubuntu 22.04+ — stačí i ten nejlevnější s 1 GB RAM (doporučujeme Contabo od 4,40 €)
  • API klíč k LLM — OpenAI, Anthropic, DeepSeek, OpenRouter nebo jakýkoliv kompatibilní provider
  • SSH přístup na váš server

Krok 1: Připojení na server

Připojte se přes SSH na váš VPS:

$ ssh root@vase-ip-adresa

Krok 2: Spuštění instalátoru

Jeden příkaz nainstaluje vše potřebné — Python 3.11, Node.js, ripgrep, ffmpeg a samotného Hermes:

# Instalace Hermes Agent $ curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash
💡 Tip: Instalátor automaticky detekuje co chybí a nainstaluje to. Jediný předpoklad je git, curl a xz-utils — na Ubuntu jsou většinou už nainstalované.

Krok 3: Obnovení shellu a spuštění

# Obnovení shellu $ source ~/.bashrc # Spuštění interaktivního chatování $ hermes

Krok 4: Konfigurace LLM providera

Při prvním spuštění vás Hermes provede konfigurací. Vyberte providera a zadejte API klíč:

# Interaktivní výběr modelu $ hermes model # Nebo vše přes průvodce $ hermes setup

Podporovaní provideři: Nous Portal, OpenRouter, OpenAI, Anthropic, Google Gemini, DeepSeek, xAI Grok a desítky dalších.

Krok 5: Nastavení Telegram bota (volitelné)

Hermes umí komunikovat přes 20+ platforem. Nejoblíbenější je Telegram:

# Konfigurace messaging gateway $ hermes gateway setup # Spuštění gateway $ hermes gateway

Gateway běží na pozadí jako systemd služba a přežije restart serveru.

Spuštění jako služba (24/7)

Aby Hermes běžel nepřetržitě, nainstalujte gateway jako systemd službu:

$ hermes gateway install $ systemctl --user enable hermes-gateway $ sudo loginctl enable-linger $USER

Minimální požadavky

💾 RAM1 GB min, 2 GB doporučeno
⚙️ CPU1 vCPU minimum
💿 Úložiště10–20 GB SSD
🐧 OSUbuntu 22.04+, Debian 12+, nebo moderní Linux
🤖 GPUNení potřeba — Hermes volá externí LLM API
🚀 Chcete ušetřit? Na naší stránce o hostingu najdete nejlevnější VPS od Contabo — od 4,40 € měsíčně s 8 GB RAM a neomezeným trafficem.

Řešení častých problémů

hermes: command not found

Obnovte shell: source ~/.bashrc nebo exec bash

Chybějící API klíč

Spusťte hermes model a vyberte providera. API klíč se uloží do ~/.hermes/.env.

Gateway se nespustí

Zkontrolujte log: tail -f ~/.hermes/logs/gateway.log
