Hermes Agent s OpenRouter

10. července 2026 · 3 min

OpenRouter je agregátor LLM API — jeden účet, jeden API klíč, přístup ke 300+ modelům od všech velkých poskytovatelů. Pro Hermes Agent je to ideální volba, protože můžete přepínat mezi modely bez změny konfigurace.

Proč OpenRouter s Hermes?

  • ✅ Jeden API klíč pro Claude, GPT, Gemini, Llama a další
  • ✅ Automatický fallback — když model selže, přepne na jiný
  • ✅ Transparentní ceny — platíte jen za použité tokeny
  • ✅ Žádné minimální zůstatky, žádné měsíční poplatky
  • ✅ Srovnání cen napříč providery na jednom místě

Krok 1: Vytvoření účtu

Jděte na openrouter.ai → klikněte Sign Up → přihlaste se přes Google/GitHub/email.

Krok 2: Získání API klíče

  1. Po přihlášení jděte do Settings → API Keys (přímý odkaz)
  2. Klikněte Create Key
  3. Pojmenujte klíč (např. "Hermes Agent")
  4. Volitelně nastavte limit útraty (doporučujeme pro začátek $10-20/měsíc)
  5. Zkopírujte klíč — začíná sk-or-v1-

Krok 3: Dobití kreditu

Jděte do Settings → Credits (přímý odkaz) → dobijte kredit (min. $5). OpenRouter strhává z kreditu podle spotřeby tokenů.

Krok 4: Konfigurace v Hermes

$ hermes model → Provider: OpenRouter → API Key: sk-or-v1-... (váš klíč) → Model: anthropic/claude-sonnet-4 (nebo jiný)

Nastavení fallback modelu

Když primární model selže (rate limit, výpadek), Hermes automaticky přepne na záložní:

# ~/.hermes/config.yaml fallback_model: provider: openrouter model: google/gemini-2.5-flash

Tipy pro úsporu

  • Levnou rutinu, drahou logiku: Na běžné dotazy používejte Gemini Flash ($0.15/M), na složité úkoly přepněte na Claude/GPT
  • Sledujte spotřebu: Na openrouter.ai/activity vidíte detailní přehled spotřeby
  • Nastavte limit: V Settings → API Keys → Edit → nastavte měsíční limit, ať neutratíte víc než chcete
💡 Doporučení: Pro začátek dobijte $10, nastavte Hermes na Gemini Flash (nejlevnější kvalitní model), a na složité úkoly ručně přepínejte na Claude Sonnet. Za $10 vydržíte při běžném používání 1-3 měsíce.
