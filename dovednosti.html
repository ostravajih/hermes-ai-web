Dovednosti (Skills)

Dovednosti jsou znalostní dokumenty, které agent načítá podle potřeby. Jsou kompatibilní s otevřeným standardem agentskills.io — používají je Claude Code, Codex, Cursor, Gemini CLI a desítky dalších nástrojů.

Jak dovednosti fungují

Dovednosti používají progresivní zveřejňování (progressive disclosure) pro minimalizaci tokenů:

Úroveň 0: skills_list() → [jméno, popis, kategorie] (~3k tokenů) Úroveň 1: skill_view(název) → Plný obsah + metadata Úroveň 2: skill_view(název, path) → Konkrétní referenční soubor

Agent načítá plný obsah dovednosti jen když ji skutečně potřebuje. Může tak mít desítky dovedností připravených s minimální zátěží kontextu.

Používání dovedností

Každá nainstalovaná dovednost je automaticky dostupná jako slash příkaz:

# V CLI nebo messaging platformě: /gif-search funny cats /github-pr-workflow create a PR for the auth refactor /excalidraw navrhni architekturu databáze /plan navrhni plán pro migraci na nový auth provider # Skládání více dovedností (až 5): /github-pr-workflow /test-driven-development oprav issue #123 a otevři PR

Formát SKILL.md

Každá dovednost je složka obsahující SKILL.md s metadaty a instrukcemi:

my-skill/ ├── SKILL.md # Povinné: metadata + instrukce ├── scripts/ # Volitelné: spustitelný kód ├── references/ # Volitelné: dokumentace ├── templates/ # Volitelné: šablony └── assets/ # Volitelné: zdroje
--- name: my-skill description: Stručný popis co dovednost dělá version: 1.0.0 platforms: [macos, linux] # Volitelné — omezit na OS metadata: hermes: tags: [python, automation] category: devops --- # Název dovednosti ## Kdy použít Spouštěcí podmínky pro tuto dovednost. ## Postup 1. Krok jedna 2. Krok dva ## Časté chyby - Známé selhání a opravy ## Ověření Jak potvrdit, že to fungovalo.

Učení nové dovednosti (/learn)

Nejrychlejší způsob jak vytvořit dovednost bez ručního psaní SKILL.md. Agent sám shromáždí materiál a vytvoří strukturovanou dovednost:

# Z lokálního adresáře /learn REST API v ~/projects/acme-sdk, zaměř se na auth a stránkování # Z online dokumentace /learn https://docs.example.com/api/quickstart # Z konverzace /learn jak jsem právě nasadil staging server

Skills Hub

Instalujte dovednosti z komunitních zdrojů:

# Vyhledání dovednosti $ hermes skills search "docker" # Instalace z hubu $ hermes skills install docker-deploy # Výpis nainstalovaných $ hermes skills list

Podporované zdroje: agentskills.io, ClawHub, LobeHub, GitHub repozitáře.

Skill bundly

Kombinace dovedností, které se často používají společně, pod jedním krátkým příkazem. Příklad:

# Místo: /github-pr-workflow /test-driven-development oprav issue #123 # Bundle "ship-it": /ship-it oprav issue #123 # Spustí obě dovednosti najednou
